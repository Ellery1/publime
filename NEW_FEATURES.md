# 新功能说明

## 已实现的5个新功能

### 1. 自动换行 ✅
**位置**: 编辑 -> 自动换行  
**功能**: 
- 可勾选的菜单项，控制文本是否自动换行
- 默认关闭（不换行）
- 应用到所有打开的编辑器标签页

**使用方法**:
1. 点击"编辑"菜单
2. 点击"自动换行"勾选/取消勾选
3. 所有编辑器会立即应用设置

---

### 2. JSON 格式化 ✅
**位置**: 编辑 -> 格式化 JSON  
**快捷键**: Ctrl+Alt+J  
**功能**:
- 自动格式化 JSON 文本
- 添加 2 空格缩进
- 保留中文字符
- 验证 JSON 格式，显示错误提示

**使用方法**:
1. 在编辑器中输入或粘贴 JSON 文本
2. 点击"编辑 -> 格式化 JSON"或按 Ctrl+Alt+J
3. JSON 会自动格式化并美化

**示例**:
```
输入: {"name":"test","value":123,"items":[1,2,3]}

输出:
{
  "name": "test",
  "value": 123,
  "items": [
    1,
    2,
    3
  ]
}
```

---

### 3. SQL 格式化 ✅
**位置**: 编辑 -> 格式化 SQL  
**快捷键**: Ctrl+Alt+S  
**功能**:
- 自动格式化 SQL 语句
- 关键字转换为大写
- 主要关键字前添加换行
- 清理多余空行

**使用方法**:
1. 在编辑器中输入或粘贴 SQL 语句
2. 点击"编辑 -> 格式化 SQL"或按 Ctrl+Alt+S
3. SQL 会自动格式化

**示例**:
```
输入: select * from users where age > 18 order by name

输出:
SELECT * 
FROM users 
WHERE age > 18 
ORDER BY name
```

---

### 4. 主题应用到主窗口 ✅
**功能**: 
- 主窗口背景色跟随当前主题
- 菜单栏样式跟随主题
- 状态栏样式跟随主题
- 标签页样式跟随主题
- 即使没有打开任何文件，主题也会正确显示

**效果**:
- 深色主题：整个窗口使用深色背景
- 浅色主题：整个窗口使用浅色背景

---

### 5. 文件对话框修复 ✅
**问题**: 点击"文件 - 打开"没有弹出对话框  
**修复**: 使用系统原生文件对话框  
**说明**: 
- 移除了自定义对话框选项
- 使用 Windows 系统原生对话框
- 更稳定可靠

**使用方法**:
1. 点击"文件 -> 打开"或按 Ctrl+O
2. 系统文件选择对话框会弹出
3. 选择文件后点击"打开"
