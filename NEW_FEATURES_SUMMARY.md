## 新增功能

### 1. 编码切换功能 ✓
**位置**: 状态栏右下角

**支持的编码**:
- UTF-8 (默认)
- GBK
- GB2312
- UTF-16
- ASCII
- ISO-8859-1

**使用方法**:
1. 打开文件后，在状态栏右下角找到"编码"下拉框
2. 选择合适的编码格式
3. 文件会自动用新编码重新加载

**注意事项**:
- 如果选择的编码无法正确解析文件，会显示错误提示并恢复到UTF-8
- 编码切换会保持当前光标位置

### 2. 语言风格切换功能 ✓
**位置**: 状态栏右下角（编码选择器左侧）

**支持的语言**:
- Plain Text (纯文本，无语法高亮)
- Python
- Java
- JavaScript
- SQL
- JSON
- Kotlin
- Markdown
- XML
- YAML

**使用方法**:
1. 在状态栏右下角找到"语言"下拉框
2. 选择想要的语言风格
3. 编辑器会立即应用相应的语法高亮

**自动检测**:
- 打开文件时会根据文件扩展名自动检测语言
- 切换标签页时会自动更新语言显示

### 3. 双击标签栏空白区域新建文件 ✓
**使用方法**:
- 在标签栏的空白区域（没有标签的地方）双击鼠标左键
- 会自动创建一个新的"未命名"文件

### 4. 鼠标中键点击关闭标签 ✓
**使用方法**:
- 在任意标签上点击鼠标中键（滚轮）
- 该标签会立即关闭
- 如果文件有未保存的修改，会弹出保存确认对话框

## 技术实现

### 状态栏组件
- 使用 `QComboBox` 实现编码和语言选择器
- 使用 `addPermanentWidget()` 将控件固定在状态栏右侧
- 通过信号槽机制响应用户选择

### 编码切换
- 使用Python的内置编码支持重新读取文件
- 捕获 `UnicodeDecodeError` 异常处理编码错误
- 保持光标位置不变

### 语言切换
- 通过 `EditorPane.set_language()` 方法切换语法高亮
- 支持的语言映射到内部语言标识符
- 不支持语法高亮的语言（Markdown、XML、YAML）设置为None

### 标签栏事件处理
- 使用 `eventFilter` 拦截标签栏的鼠标事件
- `MouseButtonDblClick` + `LeftButton` + 空白区域 -> 新建文件
- `MouseButtonPress` + `MiddleButton` + 标签上 -> 关闭标签
- 使用 `tabAt()` 方法判断点击位置
