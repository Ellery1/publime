# Publime 用户手册

欢迎使用 Publime 文本编辑器！这是一款简洁、高效的文本编辑工具。

## 快速开始

### 启动应用

双击 `Publime.exe` 即可启动应用。

**从文件关联打开**：
- 在 Windows 中设置 Publime 为默认文本编辑器
- 双击任何文本文件，会自动用 Publime 打开该文件
- **单实例模式**：如果 Publime 已经在运行，双击文件会在现有窗口中打开新标签页，而不是创建新窗口

**会话恢复**：
- 应用会自动记住上次打开的文件
- 下次启动时会自动恢复这些文件
- 如果应用异常关闭，会提示恢复未保存的内容

### 界面介绍

应用界面分为以下几个部分：

1. **菜单栏**：位于顶部，包含文件、编辑、搜索、帮助等菜单
2. **标签页栏**：显示已打开的文件，支持多标签页切换
3. **编辑区域**：主要的文本编辑区域
4. **状态栏**：位于底部，显示语言类型和编码格式

## 基本操作

### 文件操作

#### 新建文件
- 菜单：文件 → 新建
- 快捷键：`Ctrl+N`
- 双击标签页空白区域

#### 打开文件
- 菜单：文件 → 打开
- 快捷键：`Ctrl+O`
- 拖拽文件到窗口

#### 保存文件
- 菜单：文件 → 保存
- 快捷键：`Ctrl+S`

#### 另存为
- 菜单：文件 → 另存为
- 快捷键：`Ctrl+Shift+S`

#### 最近文件
- 菜单：文件 → 最近文件
- 快速打开最近编辑过的文件

### 编辑操作

#### 基本编辑
- **撤销**：`Ctrl+Z`
- **重做**：`Ctrl+Y`
- **剪切**：`Ctrl+X`
- **复制**：`Ctrl+C`
- **粘贴**：`Ctrl+V`

#### 自动换行
- 菜单：编辑 → 自动换行
- 勾选后，长行会自动换行显示

#### 代码格式化
- 菜单：编辑 → 格式化
- 快捷键：`Ctrl+Alt+F`
- 支持 JSON 和 SQL 格式化

## 搜索功能

### 当前文件查找

#### 打开查找对话框
- 菜单：搜索 → 查找
- 快捷键：`Ctrl+F`

#### 查找选项
- **区分大小写**：勾选后严格匹配大小写
- **正则表达式**：支持使用正则表达式搜索

#### 查找操作
- **查找下一个**：查找下一个匹配项
- **查找上一个**：查找上一个匹配项
- **替换**：替换当前匹配项
- **全部替换**：替换所有匹配项

### 跨文件搜索

#### 打开跨文件搜索
- 菜单：搜索 → 在文件夹中查找
- 快捷键：`Ctrl+Shift+F`

#### 搜索模式

**1. 搜索文件内容**（默认开启）
- 在文件内容中搜索指定文本
- 适合查找代码片段、函数名等

**2. 搜索文件名**
- 在文件名中搜索指定文本
- 适合快速定位文件

**3. 同时搜索**
- 可以同时勾选两种模式

#### 匹配选项

**模糊匹配**（默认）
- 文件名包含搜索文本即可
- 例如：搜索 "test" 可以找到 `test.py`, `my_test.txt`

**精确匹配**
- 文件名必须完全等于搜索文本
- 例如：搜索 "test.py" 只能找到 `test.py`

#### 其他选项
- **区分大小写**：是否区分大小写
- **正则表达式**：使用正则表达式搜索（仅对文件内容有效）
- **文件类型**：过滤特定类型的文件，如 `*.py, *.txt`

#### 使用示例

**示例 1：查找所有 Python 文件**
1. 勾选"搜索文件名"
2. 文件类型输入：`*.py`
3. 点击"搜索"

**示例 2：查找包含某个函数的文件**
1. 勾选"搜索文件内容"
2. 输入函数名
3. 点击"搜索"

**示例 3：查找特定文件**
1. 勾选"搜索文件名"
2. 勾选"精确匹配"
3. 输入完整文件名
4. 点击"搜索"

#### 查看结果
- 搜索结果分为"文件名匹配"和"文件内容匹配"两部分
- 双击结果可以打开文件并跳转到对应位置

## 高级功能

### 语法高亮

应用会自动检测文件类型并应用语法高亮，支持：
- Python
- Java
- JavaScript
- SQL
- JSON
- Kotlin
- Markdown

#### 手动切换语言
在状态栏右侧的"语言"下拉框中选择语言类型。

### 编码选择

在状态栏右侧的"编码"下拉框中选择文件编码：
- UTF-8（默认）
- GBK
- GB2312
- UTF-16
- ASCII
- ISO-8859-1

切换编码后会自动重新加载文件。

### 主题切换

#### 切换主题
- 菜单：帮助 → 切换主题
- 快捷键：`Ctrl+Shift+T`

支持深色和浅色两种主题。

### 大文件处理

- 打开超过 50MB 的文件时会显示警告
- 超过 10MB 的文件会自动禁用语法高亮以提高性能
- 支持打开和编辑大型日志文件

### 崩溃恢复

应用每 5 分钟自动保存一次所有打开的文件：
- 如果应用意外关闭，下次启动时会提示恢复
- 可以选择恢复或放弃之前的会话

### 会话恢复

应用会自动记住上次打开的文件：
- 正常关闭应用时，会保存当前打开的所有文件列表
- 下次启动时自动恢复这些文件
- 只恢复已保存的文件（未命名的新文件不会恢复）

### 文件关联

可以将 Publime 设置为默认文本编辑器：
1. 右键点击任意文本文件
2. 选择"打开方式" → "选择其他应用"
3. 浏览并选择 `Publime.exe`
4. 勾选"始终使用此应用打开 .txt 文件"
5. 以后双击文本文件就会自动用 Publime 打开

## 常见问题

### 如何打开多个文件？

可以通过以下方式打开多个文件：
1. 多次使用"打开文件"功能
2. 拖拽多个文件到窗口
3. 从"最近文件"菜单打开

每个文件会在新标签页中打开。

### 如何关闭标签页？

点击标签页上的 ✕ 按钮，或者右键点击标签页选择关闭。

如果文件有未保存的修改，会提示是否保存。

### 文件编码显示乱码怎么办？

1. 在状态栏右侧的"编码"下拉框中选择正确的编码
2. 常见中文编码：GBK、GB2312
3. 国际通用编码：UTF-8

### 如何搜索整个项目？

1. 按 `Ctrl+Shift+F` 打开跨文件搜索
2. 选择项目根目录
3. 勾选"搜索文件内容"
4. 输入搜索内容
5. 点击"搜索"

### 如何格式化 SQL 或 JSON？

1. 打开 SQL 或 JSON 文件
2. 按 `Ctrl+Alt+F` 或选择菜单"编辑 → 格式化"
3. 应用会自动检测文件类型并格式化

### 应用崩溃了，文件会丢失吗？

不会。应用有自动保存功能：
1. 每 5 分钟自动保存一次
2. 下次启动时会提示恢复
3. 可以选择恢复之前的会话

### 如何让 Publime 记住打开的文件？

应用会自动记住：
1. 正常关闭应用时，会保存当前打开的文件列表
2. 下次启动时自动恢复这些文件
3. 无需手动操作

### 如何用 Publime 打开文件？

有多种方式：
1. **双击文件**：将 Publime 设置为默认编辑器后，双击文件即可打开
2. **拖拽**：直接拖拽文件到 Publime 窗口
3. **菜单**：使用"文件 → 打开"菜单
4. **快捷键**：按 `Ctrl+O` 打开文件对话框
