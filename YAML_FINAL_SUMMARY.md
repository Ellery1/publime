## 关键改进点

### 1. 去除下划线
- **问题**：键名显示有下划线
- **原因**：使用了 `markdown_link` 格式
- **解决**：改用 `class` 格式（青色粗体，无下划线）

### 2. 布尔值独立颜色
- **需求**：布尔值要与数字区分开
- **实现**：布尔值使用 `keyword` 格式（粉红色），数字使用 `number` 格式（紫色）
- **效果**：粉红色 vs 紫色，对比鲜明

### 3. 时间键名独立颜色
- **需求**：时间相关键名要与普通键名区分开
- **实现**：时间键名使用 `string` 格式（橙黄色），普通键名使用 `class` 格式（青色）
- **匹配规则**：包含 "date" 或 "time" 的键名（不区分大小写）
- **效果**：橙黄色 vs 青色，一目了然

## 技术实现

```python
# 1. 注释（最高优先级）
# 2. 文档分隔符
# 3. 时间键名（必须在普通键名之前！）

# 5. 布尔值和 null
# 6. 数字
# 7. 字符串值
# 8. 列表标记
# 9. 锚点和别名
```

- `number` → 紫色 #AE81FF
