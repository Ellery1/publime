# Publime 发布说明

## 版本信息

**版本**: 1.0.0  
**发布日期**: 2026-01-18  
**文件大小**: 44 MB  
**平台**: Windows 7 及以上

## 新特性

### 核心功能
✅ **多标签页文件管理** - 同时编辑多个文件，轻松切换  
✅ **语法高亮** - 支持 Python, Java, JavaScript, SQL, JSON, Kotlin, Markdown  
✅ **智能搜索** - 支持文件内容和文件名双模式搜索  
✅ **代码格式化** - 一键格式化 JSON 和 SQL 代码  
✅ **主题切换** - 深色/浅色主题，保护眼睛  
✅ **崩溃恢复** - 自动保存，永不丢失数据  

### 高级搜索功能
- **文件内容搜索**: 支持正则表达式，快速定位代码
- **文件名搜索**: 模糊匹配或精确匹配
- **文件类型过滤**: 只搜索指定类型的文件
- **搜索结果分组**: 清晰展示匹配结果

### 用户体验
- **拖放支持**: 直接拖拽文件到窗口打开
- **最近文件**: 快速访问最近编辑的文件
- **大文件优化**: 支持 50MB+ 大文件
- **多编码支持**: UTF-8, GBK, GB2312 等
- **自动换行**: 长行自动换行显示

## 快捷键

### 文件操作
- `Ctrl+N` - 新建文件
- `Ctrl+O` - 打开文件
- `Ctrl+S` - 保存文件
- `Ctrl+Shift+S` - 另存为

### 编辑操作
- `Ctrl+Z` - 撤销
- `Ctrl+Y` - 重做
- `Ctrl+X/C/V` - 剪切/复制/粘贴

### 搜索操作
- `Ctrl+F` - 查找
- `Ctrl+Shift+F` - 在文件中查找

### 其他
- `Ctrl+Alt+F` - 格式化代码
- `Ctrl+Shift+T` - 切换主题

## 安装说明

### 方式 1：直接运行（推荐）
1. 下载 `Publime.exe`
2. 双击运行即可使用
3. 无需安装，绿色便携

### 方式 2：从源码运行
1. 安装 Python 3.9+
2. 克隆仓库
3. 安装依赖：`pip install -r requirements.txt`
4. 运行：`python main.py`

## 系统要求

- **操作系统**: Windows 7 或更高版本
- **内存**: 建议 2GB 以上
- **磁盘空间**: 约 100MB

## 已知问题

- 超大文件（>100MB）可能导致性能下降
- 某些特殊编码的文件可能显示乱码（请手动选择编码）

- **UI 框架**: PySide6 (Qt for Python)
- **打包工具**: PyInstaller
- **测试框架**: pytest
- **语言**: Python 3.10
